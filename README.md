# yolov5-deepsort-tensorrt

## Requirement
1. RTX2060(notebooks)
2. cuda10.2
3. cudnn8.0.4
4. TensorRT-7.1.3.4
5. nvidia-driver-510
6. anaconda3
7. opencv4.5.3
8. cmake3.21.0

## Build&&run
1. clone this project
2. mkdir build
3. cmake ..
4. make
5. ./deepsort

## Create yolo.engine and deepsort.engine
1.yolo.engine

follow https://github.com/wang-xinyu/tensorrtx.git

2.deepsort.engine

follow https://github.com/ZQPei/deep_sort_pytorch.git
